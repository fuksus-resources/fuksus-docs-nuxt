---
title: General config
description: This section provides an overview of Quick Start Guide.
---

::warning
  We strongly recommend adding the image path as your inventory image path. 
  Example with ox_inventory, the path would be `nui://ox_inventory/web/images`
::

```lua
Config = {}

Config.Locale = 'en'
Config.DrawDistance = 30.0
Config.MarkerDistance = 5.0

Config.ImageLocation = 'nui://ox_inventory/web/images/'
Config.CraftingTickSpeed = 500

Config.Exp = {
    Exponent = 1.3,
    Basexp = 1250,
    AddXpForCraft = 100
}

Config.SaveCraftingProgress = true
Config.SaveTime = 15

Config.Target = 'ox_target'

Config.Debug = false

-- Global marker
Config.Marker = {
    type = 0,
    scale = {
        x = 0.5,
        y = 0.5,
        z = 0.5
    },
    color = { r = 99, g = 102, b = 241, a = 100 },
    offset = {
        x = 0.0,
        y = 0.0,
        z = 0.5
    },
    face = true,
    bobUpDown = true,
    spin = false
}
```
